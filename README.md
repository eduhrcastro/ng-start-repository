## Requirements

- [Angular.js](http://angularjs.org)

## Build

```
$ yarn install
$ gulp build
```

## Tests

- Unit:
 - Uses [Karma](http://karma-runner.github.io) + [Jasmine](http://jasmine.github.io/)
 - Files: `specs/*.spec.js`

```
$ gulp test
```
